can_process?( data )

Returns true if the node has a Recv edge that can process this data. Only one recv edge should be valid for a certain input, and if that‘s not the case then they‘d better be equivalent or the FSA was created incorrectly. :)

current_node?()

True if this node is the current node in the FSA

invoke_recv_edge( recv_edge, data )

Invoke the action block of a Recv edge, returns the new current_node of the FSA. Usually called by the parent FSA as part of deliver

invoke_send_edge( send_edge )

Invokes the action block of a given Send edge. Returns the output that should be sent.
