

REQUIREMENTS:
------------

step 1: Python runtime environment
    Python
        http://python.org/ftp/python/2.6.2/python-2.6.2.msi
    easy_install
        http://pypi.python.org/packages/2.5/s/setuptools/setuptools-0.6c9.win32-py2.5.exe

step 2: Python packages
    SQLAlchemy
        $ easy_install sqlalchemy
    psycopg2
        download -> http://initd.org/pub/software/psycopg/psycopg2-2.0.12.tar.gz
        XXX - best to just package this, it'll never work otherwise.

INSTALLATION
------------

The runtracer must be installed into the C:\runtracer\ directory. This can be
done by moving the unpacked source code into the correct directory.
