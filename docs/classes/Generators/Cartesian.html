Class Generators::Cartesian
In: src/generators.rb
Parent: Generator

In: n Series where n>0. Out: Arrays, each having n elements

The Cartesian generator will output each item in the cartesian product of the series passed as arguments. Output is in the form of an array.

Example: 

 require 'generators'
 g=Generators::Cartesian.new( ('a'..'c'), [1,2], %w( monkey hat ) )
 while g.next?
      foo, bar, baz=g.next
      puts "#{foo} : #{bar} -- #{baz}"
 end

Produces: 

 a : 1 -- monkey
 a : 1 -- hat
 a : 2 -- monkey
 a : 2 -- hat
 b : 1 -- monkey
 b : 1 -- hat
 b : 2 -- monkey
 b : 2 -- hat
 c : 1 -- monkey
 [etc]

Note: The cartesian product function is quite forgiving and will take Generators, Ranges and Arrays (at least)
